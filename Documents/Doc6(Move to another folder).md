### how to move a file to another folder?

<img width="720" height="1341" alt="File tree opened in Beyoneer" src="https://github.com/user-attachments/assets/bcc154ff-8ae0-4d09-a06b-cf7b02fef4f2" />

- Open Beyoneer.
- Click on menu.
- select your file you want to move to another folder.
- long press (for mobiles or right click for desktops).
- file tree opens.
- select move to folder.
- type your folder name or create via path you writes ( its automatically creates a folder if written in path or move the file to existing folder in path. )
