### How to rename files in Beyoneer IDE?

<img width="720" height="1600" alt="Renaming file on Beyoneer ide" src="https://github.com/user-attachments/assets/65f793a6-2234-4f33-aad1-d0b67eb57ab0" />


- Open Settings (≡).
- Long press on file you want to rename it (Mobiles/touch) or right click(Computer).
- Ui tree came on your screen select option rename.
- Rename your file and click on OK.
