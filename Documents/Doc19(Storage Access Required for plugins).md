### What does this means? (explained below)
<img width="720" height="1316" alt="Storage Access permission required for some plugins" src="https://github.com/user-attachments/assets/bdc0d7a7-c5c6-4a35-9f84-dc4a40532a59" />

This Popup dialogue box means, the plugin you are going to **activate** needs your Beyoneer IDE Browser storage **permission.**
#### This means the plugin can access:
- Your Indexdb(Browser Storage) data.
- Every Files/Folders/Projects/Binaries and all formats supported/un supported, plugin can access after you click the button  " **Ok, I Trust it.** "

  #### Is it safe for my projects?
  We recommed you to use our [**Marketplace**](https://beyoneer.xyz/marketplace) for most trusted plugins, that are approved or added by MagmaMinesTeam or Beyoneer IDE Staffs.
Plugins that are approved/added by Beyoneer is more trustable & provides direct community support with plugin creators.

### What if a random plugin stoles my data/codes?
- Report to the Beyoneer IDE on discord.
- If you used an untrusted plugins or plugins that are not listed on official marketplace, we try to ban the person but not **100%** guaranteed for a random plugin.
- If the code stealer has a discord account, you can give It's Discord ID/username to the staff, we try to announce his/her name with very respectful words with chears from our small community 😊.
